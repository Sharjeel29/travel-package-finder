# ✈️ Travel Package Finder

## Overview
Travel Package Finder is a full-stack web application where users can explore travel packages, create bookings, and manage their profiles.

## Features
- User registration and login system  
- Create and manage user profile  
- Book travel packages  
- View and delete bookings  
- REST API built with NestJS  
- Frontend built with Next.js  

## Tech Stack

### Frontend
Next.js • React • JavaScript

### Backend
NestJS • PostgreSQL • REST API

### Tools
Postman • pgAdmin

## Project Structure
frontend/ – Next.js frontend  
backend/ – NestJS backend server
